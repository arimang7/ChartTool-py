# 하모닉 트레이딩 #2: AB=CD

**작성자**: 김오르내림 (@olnlkim)  
**주제**: 하모닉 트레이딩의 가장 기본 구조인 AB=CD 패턴

## 핵심 메시지
- 피보나치 비율 자체가 틀린 것이 아니라 → **구조** 없이 숫자만 사용하는 것이 문제다.
- AB=CD는 **D 지점을 정확히 맞히는 도구가 아니라**, D 지점(완성 구간)에서 **반응/반전 가능성을 관찰하기 위한 프레임**이다.
- 가장 중요한 포인트는 **D가 아니라 C**다. C의 되돌림 깊이가 CD의 최소 완성 목표를 결정한다.

## AB=CD란?

- **Bullish AB=CD** : 상승 구조  
  A(저점) → B(고점) → C(저점) → D(고점 예상)
- **Bearish AB=CD** : 하락 구조  
  A(고점) → B(저점) → C(고점) → D(저점 예상)

**핵심 성격**
- AB 레그와 CD 레그가 **길이·비율적으로 대칭**을 이루는 구조
- D는 **minimum price objective**(최소 가격 목표) 구간
- 가격이 D 구간에 도달하면 **반응(reaction)** 또는 **반전(reversal)** 가능성 ↑

## AB=CD 이해의 출발점: 스윙 하이/로우 고정

1. **다우이론 기반**으로 주요 스윙 포인트 결정
   - 상승 추세: 고점·저점 점점 높아짐
   - 하락 추세: 고점·저점 점점 낮아짐
2. **스윙 하이(Swing High)** = 방향 전환을 드러내는 주요 고점
3. **스윙 로우(Swing Low)** = 방향 전환을 드러내는 주요 저점

→ 모든 고점/저점이 아니라 **추세를 정의하는 구조적 포인트**만 사용

### Bullish AB=CD 순서 예시
- A = 첫 번째 주요 스윙 로우
- B = 구조를 전진시키는 스윙 하이
- C = 되돌림 후 다시 방향 전환하는 스윙 로우
- D = 예상 목표 (미래 스윙 하이)

## 실전 작도 순서 (가장 중요한 부분)

실제 트레이딩에서는 **거꾸로** 진행합니다.

1. A → B 레그 고정
2. B → C 되돌림 확인 (주로 0.382 ~ 0.886 사이)
3. **BC 레그를 기준으로 확장(Extension) 그려서 D 후보 먼저 표시**
4. 그 다음 **AB 레그와 동일 길이(1:1) 투영**해서 AB=CD 완성점 확인
5. 두 구간이 **겹치는 범위 = PRZ** (Potential Reversal Zone)

### C 되돌림 깊이별 대표 D 후보 (BC 확장값)

| C 되돌림 비율 | 대표적인 BC 확장 D 후보 | 비고                             |
|---------------|--------------------------|----------------------------------|
| 0.382         | 2.618                    | 과열 확장 가능성 (추가 레벨 열림) |
| 0.500         | 2.000                    | 전형적 대칭                     |
| 0.618         | 1.618                    | 가장 전형적인 AB=CD              |
| 0.707         | 1.414                    | 자주 연결됨                      |
| 0.786         | 1.272                    | 안정적 반복                      |
| 0.886         | 1.130                    | 깊은 되돌림 → 가까운 반응        |

→ **C가 힌트를 준다** → BC 확장으로 D 후보를 먼저 좁힌다 → AB=CD로 검증

## PRZ는 한 점이 아니라 구간(겹침)

PRZ =  
- AB=CD 완성점 (AB 길이 1:1 투영)  
+ BC 확장 레벨 (1.13, 1.272, 1.414, 1.618, 2.0, 2.618 등)  
→ 이 둘이 **가격대에서 겹치는 구간**

가격이 PRZ에 도달했을 때  
→ 캔들 반응, 구조 붕괴 여부, 되돌림 형태 등을 **관찰** → 판단

## AB=CD 형태 구분

- **Standard AB=CD** : AB ≈ CD (1:1에 가까움)
- **Alternative AB=CD** : CD > AB (주로 1.27배 또는 1.618배 확장)

## AB=CD가 깨지는 경우

- C 지점 애매함
- CD의 속도·성격이 AB와 너무 다름
- PRZ를 반응 없이 관통
→ 이런 경우 무리하게 구조를 우기지 말고 **패턴 실패**로 인정

## 오늘의 핵심 흐름 요약

1. C의 되돌림 깊이 확인  
2. BC 확장으로 D 후보 구간 먼저 그림  
3. AB=CD (1:1) 구조로 검증  
4. 겹치는 구간 = PRZ  
5. PRZ 도착 시 → **반응 관찰** → 진입 여부 판단

→ AB=CD는 **진입 신호**가 아니라 **반전 가능성이 높은 구간을 좁혀주는 프레임**

## 마무리

- 글로 읽는 것과 실제 차트에서 여러 번 작도해보는 것 사이에는 큰 간극이 있다.
- 다양한 종목 + 다양한 타임프레임에서 직접 그려보는 연습이 필수
- 예시 조합 (C / D 목표)
  - 0.886 → 1.13 Bullish
  - 0.500 → 2.0  Bearish
  - 0.786 → 1.27  Bullish
  - 0.618 → 1.618 Bearish
  - 등등

**AB=CD의 완성점(1.0)은 정답이 아니라 최소 완성점** → 초과 움직임은 흔하며, 허용 범위는 이후 학습에서 다룸.

하모닉은 숫자를 외우는 게 아니라 **구조를 읽는 기술**입니다.
